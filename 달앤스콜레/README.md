- `달앤스콜레_수강생관리.xlsx` — 기수별 수강생 출석·과제·시험 현황

## 운영 규칙

- 수업자료는 강별 폴더에 pptx + pdf 함께 보관
- 수강생 과제는 `기수별/N기/` 폴더에 학생명으로 구분
- 시험지는 배포용·정답용 구분하여 보관
