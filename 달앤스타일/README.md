# 달앤스타일 시스템

인테리어 시공·스타일링 프로젝트 운영 시스템입니다.

## 폴더 구조

```
🏗️달앤스타일/
├── 진행중현장/
│   └── {연도}_{현장명}/      # ex) 2026_송파가락쌍용
│       ├── 도면/
│       ├── 사진/
│       ├── 견적서/
│       └── 계약서/
├── 완료현장/
│   └── {연도}_{현장명}/
├── 협력업체DB/
│   └── 달앤스타일_업체연락처DB.xlsx
├── 견적서_계약서/
│   └── {연도}/
└── 도면/
    └── {연도}/
```

## 현재 진행 현장 (2026년 5월 기준)

| 현장 | 단계 | 주요 이슈 |
|---|---|---|
| 송파 가락쌍용 | 마무리 | 전기작업, 폴딩도어 결제 필요 |
| 옥주부사옥 (정종철) | 마무리 | 악세서리·콘센트 설치 |
| 압구정 미성아파트 | 초기 | 자재 발주, 예림 서류 |
| 이시영님댁 | 납품대기 | 옌센 침대 6/4 배송 |

## 관리 파일

- `달앤스타일_현장관리표.xlsx` — 전체 현장 단계 현황 + 결제 추적
- `달앤스타일_업체연락처DB.xlsx` — 협력업체 계좌·연락처 DB

## 새 현장 생성 절차

1. `진행중현장/` 에 `연도_현장명` 폴더 생성
2. 내부에 `도면`, `사진`, `견적서`, `계약서` 폴더 생성
3. `달앤스타일_현장관리표.xlsx` 에 행 추가 후 단계 기록

## 현장 완료 처리

1. `진행중현장/` → `완료현장/` 폴더 이동
2. 현장관리표 상태 업데이트
3. 최종 사진 `사진/완성본/` 폴더에 정리

## 운영 규칙

- 바탕화면에는 현재 작업 중인 파일만 보관 (작업 완료 시 즉시 해당 폴더로 이동)
- 결제 계좌·금액 확인은 업체DB 파일 기준 (카카오톡 채팅 의존 금지)
- 현장 주요 결정사항은 현장관리표 메모란에 기록
