# 달앤 통합 업무 시스템

달앤스콜레(교육)와 달앤스타일(인테리어 시공·스타일링) 두 사업의 파일 관리, 현장 운영, 수강생 관리를 위한 통합 시스템입니다.

## 구조 개요

```
Desktop/달앤/
├── 📚달앤스콜레/        # 교육 사업
├── 🏗️달앤스타일/       # 시공·스타일링 사업
└── 📋공통/             # 공통 서류
```

## 팀 구성

| 이름 | 역할 |
|---|---|
| 달앤지현 | 대표 |
| 달앤 박장후과장 | 현장 총괄 실장 |
| 달앤강문주 | 코디네이터 (도면·자재·서류) |
